How to Use

  • From OSCAR, navigate to the Admin page and choose "Report by Template".
  • Select "OSCAR eForm Query" from the list of reports and run it. Refer here for the Report By Template if you do not have it
  • After running the report, scroll to the bottom and click "Export as CSV" to save the file
  • Import the saved file into this page and the reformatted file will appear as a download for analysis in Excel.
  • **This tool only works with comma-separated (*.csv) files

This tool is safe for use with patient health information (PHI). All data processing is done on your local computer. Please take steps to protect the PHI you extract from OSCAR.
